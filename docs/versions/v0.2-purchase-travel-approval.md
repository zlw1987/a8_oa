# V0.2 Purchase, Travel, And Approval

## Scope

This version introduced the main request workflows and reusable approval engine.

## Delivered Capabilities

- Purchase request creation, update, submit, approve, return, reject, cancel, actual spend, and closeout.
- Travel request creation, itinerary, estimated expenses, per diem totals, submit, approve, return, reject, cancel, actual expenses, and closeout.
- Approval rules by request type, department, amount range, effective dates, and priority.
- Approval steps for specific users, department managers, requester managers, department approvers, finance, purchasing, and global approvers.
- Approval task pool behavior with claim, release, candidate tracking, and assigned task views.
- Approval dashboards for current tasks, history, due state, reminders, and escalation handling.
- Requester self-approval prevention for approval tasks.

## Key Apps

- `approvals`
- `purchase`
- `travel`
- `dashboard`

## Regression Coverage

- Purchase and travel submit workflows.
- Pool task claim/release flows.
- Approval history and filtering.
- Cross-request dashboard behavior.
- Duplicate submit prevention.
- Returned request edit and resubmit flows.
