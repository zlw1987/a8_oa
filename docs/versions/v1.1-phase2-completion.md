## Delivered Scope

### Line-Level Receipt Matching

- PR detail actual expense lines show receipt and invoice status.
- TR detail actual expense lines show receipt and invoice status.
- Users with permission can upload receipt/invoice support directly to an actual expense line.
- Users with permission can link an existing request-level attachment to a specific actual expense line.
- Missing receipt review detail points to the exact actual expense line.
- Request-level attachments remain available for quotes and general support.

### Direct Project Cost Policy

- Direct Project Cost Policy list/create/edit pages exist.
- Direct Project Cost Policies are available from Setup navigation and System Setup.
- Company card allocation to Project Direct Cost applies the configured policy.
- Policy results supported:
  - `ALLOW`
  - `REVIEW`
  - `REQUIRE_PROJECT_OWNER_APPROVAL`
  - `BLOCK`
- `REVIEW` and `REQUIRE_PROJECT_OWNER_APPROVAL` create accounting review items.
- `BLOCK` prevents direct project cost allocation.
- Card Transaction Detail shows direct project cost policy result and project owner review status.

### Approval Delegation

- My Delegations page exists.
- Create/Edit Delegation pages exist.
- Active Delegations admin/finance view exists.
- Delegated tasks appear in My Tasks during the active date range.
- Delegated approval writes history showing the delegate acted on behalf of the original approver.
- Requester self-approval remains blocked.

### Approval Reassignment And Overdue Handling

- Finance/Admin users can reassign open approval tasks with a required reason.
- My Tasks shows overdue assigned and pool task counts.
- Overdue tasks remain visually highlighted.
- `process_approval_escalations` command is available as an operational alias for the escalation process.
