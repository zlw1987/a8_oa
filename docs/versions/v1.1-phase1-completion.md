## Delivered Scope

### Accounting Period / Month-End Close

- Accounting Period list/create/detail.
- Current open period visibility.
- Period close checklist.
- Close period action with notes.
- Reopen period action with reason.
- System Setup shortcut.
- Finance navigation entry.

### Financial Locking

Closed accounting periods block ordinary financial changes:

- PR actual spend posting.
- TR actual expense posting.
- Company card transaction creation.
- Company card allocation changes.
- Refund / credit / reversal entry.
- Direct reopen/correction for requests whose request date belongs to a closed period.

### Refund / Credit / Reversal

- PR detail page supports Record Refund / Credit / Reversal.
- TR detail page supports Record Refund / Credit / Reversal.
- Refund/credit is recorded as a separate negative actual entry.
- Original actual expense remains visible.
- Refund can link to the original actual line.
- Negative consume ledger entry reduces consumed budget.

### Reopen / Correction

- Closed PR can be reopened by Finance/Admin with reason.
- Closed TR can be reopened by Finance/Admin with reason.
- Correction reference is supported.
- Correction status is visible on request detail.
- Reclose uses existing closeout validation.
- Closed-period request correction is blocked and should be handled through controlled adjustment flow.
