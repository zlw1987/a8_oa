# V0.1 Foundation

## Scope

The foundation version established the base Django project and core organization model.

## Delivered Capabilities

- Custom user model for OA users.
- Department model with department type, manager, hierarchy, cost center, and active state.
- User-department links for primary department, approval responsibility, and visibility responsibility.
- Shared request and budget choice enums.
- Base Django settings, URL routing, templates, static/media wiring, and local email file backend.

## Key Apps

- `accounts`
- `common`
- `dashboard`

## Operational Notes

- Local development uses SQLite.
- The default local email backend writes generated messages into `sent_emails`.
- Auth redirects are configured for the built-in Django auth views.
