# V1.2A Finance Report Drill-Down

## Status

V1.2A is implemented.

This release enhances Finance Reports drill-down while keeping the V1.1 Phase 1, Phase 2, and Phase 3 business-control workflows closed and unchanged.

## Delivered Scope

- Department Spending Summary rows now link to a department drill-down detail page.
- Department drill-down shows related projects, purchase requests, travel requests, purchase actual spend, and travel actual expenses.
- Monetary values use the existing shared money formatter.
- Direct URL access to department drill-down uses the same accounting/report permission guard as Finance Reports.
- Existing Finance Reports drill-down links are preserved:
  - Project Budget Summary to project budget ledger.
  - Open Reserve rows to PR/TR detail.
  - Over-Budget Exceptions and Accounting Review Aging rows to Accounting Review Detail.
  - Unmatched Card rows to Card Transaction Detail.

## Limitations

- Drill-down remains link/detail based, not an advanced BI framework.
- Excel export is not implemented.
- Saved filters are not implemented.
- Date, department, and project report filters are not implemented yet.
- No i18n work was included in V1.2A.
- No models, migrations, or V1.1 Phase 1/2/3 workflow changes were added.
