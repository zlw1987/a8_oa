# V0.3 Budget And Project Ledger

## Scope

This version connected purchase and travel requests to project budget control.

## Delivered Capabilities

- Project master data with budget amount, owner, department, type, and lifecycle status.
- Budget reservation on request submit.
- Budget release on request cancel, return, or reject.
- Budget consumption when actual spend or actual travel expense is recorded.
- Partial reserve release when actual consumption is below the approved estimate.
- Project budget ledger that shows purchase and travel entries together.
- Project summary helpers for reserved, consumed, released, and available amounts.
- Project access checks for related requesters and owners.

## Key Apps

- `projects`
- `purchase`
- `travel`

## Regression Coverage

- Purchase and travel ledger entries.
- Project ledger page access.
- Ledger summary consistency with project helpers.
- Budget release behavior for return, reject, cancel, and actual closeout.
